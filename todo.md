# 연구 계획

## 하고 싶은 것
산업용 CT/X-ray 이미지에서 노이즈를 제거하고 싶다.
현재 장비로는 1번 스캔밖에 못하는데, 10번 스캔한 것처럼 깨끗한 이미지를 얻고 싶다.

## 문제 상황
- 단일 스캔 이미지는 노이즈가 심해서 결함 검출 정확도가 낮다
- 반복 스캔은 비용과 시간 문제로 불가능하다
- 기존 알고리즘 기반 denoising은 실제 노이즈 특성을 제대로 반영하지 못한다
- 결함 자체는 제거하면 안 되고, 노이즈만 제거해야 한다

## 원하는 결과
PSNR 30dB 이상을 달성하고, 가능하면 실시간 처리도 되면 좋겠다.
성능이 우선이지만, 경량 모델이면 더 좋다.

## 제약 조건
- 모델 크기와 inference time 중요
- 성능 및 경량화 tradoff에서 성능 우선

## 측정 지표
PSNR, SSIM
